Recommendations

  • If you look at the code for this site, don't get discouraged. There's a lot of formatting and JavaScript I put in to make the site usable. The code of the site is not intended to be the tutorial themselves. Do feel free to look and ask questions though.
  • Look at the code from other sites using your browser. Press <CNTRL + SHIFT> + i to open the developer tools panel and look at the code there under the Sources tab . You can see network calls it makes under the Network tab, find HTML elements in a crowded page using the Elements tab, and see error messages and developer output under the Console tab.
  • Download and use VSCode as your IDE (Integrated Development Evironment). You can use anything you want really (I started learning using the VI editor and a browser) but VSCode is widely used and has a plethora of plugins that can make your life significantly easier.
  • Pick a browser to test on that is highly used but don't get religious about it. Different browsers will behave differently from time to time. It will frustrate the heck out of you but press on.
  • Create a GitHub account
  • Create a Vercel account
  • Create a CodeSandbox account
  • Search on YouTube for various coding topics. Subscribe to channels you like.
With these bits you can do things like:
  • Use it:
    1. Create and test a website in your IDE and browser.
    2. Upload your project to GitHub.
    3. Deploy your project to Vercel (for free!) to allow others to use it.
  • Show off your skills:
    1. Create and test a website in your IDE and browser.
    2. Upload your project to GitHub.
    3. Deploy your project to CodeSandbox (also for free!) to demo it in realtime online with others.
