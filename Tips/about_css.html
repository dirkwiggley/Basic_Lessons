About CSS

CSS stands for Cascading Style Sheets. The Cascading part refers to the fact that css elements are attached to HTML tags and every HTML from that tag and all of it's children will have the same css attached to them by default. The simplest way to 'turn off' css in a child HTML element is to give that element css settings of it's own.

A good resource for CSS is W3 Schools.

A few things to consider about CSS:

  • CSS no longer just a style tool. It has some very powerful features that you can use. Browsing YouTube is a good way to find and make use of some of these.
  • There are three common ways to add css styles to a document
    1. Inline Style - example:

      <p style="color:red;">This is a paragraph.</p>

      produces this:

      This is a paragraph

    2. External style - In this case you create a separate css file that you include in your html. You then add that styling to your tags using the class attribute. It looks like this

      In a new file (i.e. mystyle.css)

              h1 {
          margin: 0px;
          padding: 1em;
        }
        
        .myFooter {
          position: fixed;
          bottom: 0;
          background-color: #F1F1F1;
          min-width: 100%;
          padding-left: 15px;
        }

      In a css file with that, any h1 tag will automatically have a margin of 0 pixels and padding of 1em. Also any tag that you give the class="footerClass" attribute to will have the styles listed above. You would use that like this:

      3.           <head>
            <link rel="stylesheet" href="mystyle.css">
          </head>
          <body>
            <footer class="myFooter">stuff in footer</footer>
          </body>
    3. Internal style is where you put the contents of a stylesheet inside the file with the HTML that uses it. It looks like this: 
                <head>
          <style>
            body {
              background-color: linen;
            }
            
            h1 {
              color: maroon;
              margin-left: 40px;
            }
          </style>
          </head>

      And it is used in exactly the same way in your HTML tags as in the External Style example above.

The previous list is intended to just suggest a starting point for studies into css. Any basic css tutorial will cover them in much more detail.
