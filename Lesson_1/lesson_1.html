Lesson 1

Welcome to Lesson 1. Here we will create a simple HTML page in the VSCode IDE (Integrated Development Environment) and display it in a browser. We'll also examine a feature or two of the browser. Lets begin.

Open VSCode and create a new file named index.html using the command bar command: File => New File => <file name> (in this case the file name is index.html). The name could be just about anything as long as it has an .htm or .html file extension. The name index.html is the name used as a standard entry point for web sites.
